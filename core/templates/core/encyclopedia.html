Encyclopedia Galactica

 


This was the first project for the Code Institute course. It consists of nothing but html and css.

I chose this subject since Dr Asimov is easily my favorite sci-fi author.

As the first project, it is not perfect. There are things that I would do differently in retrospect but overall I am decently satisfied with the result considering I had not touched html and css in years before doing this.

  • The color contrasts could be a lot better.
  • The timeline page is a bit heavy and could do with some splitting up.
  • The inline video works, but could use some work.
  • Removing the scroll-bars so it does not like the old frames-tag would be good too.
  • Not to mention, the color scheme on the "popup" you get in the upper-right corner on lower resolutions is not ideal.
