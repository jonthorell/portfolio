Music Quiz

 


This was the second project for the Code Institute course. It is essentially a quiz game to determine if you have any musical taste

The game is written entirely in javascript. There are, of couse, some html and css involved as well but that is "merely" to make it look good (in the case of css) and to give the javascript a DOM to manipulate (in the case of html) to provide feedback to the user.

All game logic takes place in the javascript file.

People from Sweden might think it looks familiar. It was not intentionally, but some of the user interface does look like the mobile game "Quizkampen". But without the ability to compete against someone else. This is a one-player game only.
