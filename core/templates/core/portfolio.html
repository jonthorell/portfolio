Portfolio

 


This was created when the Code Institute course was finished. Basically, just a very simple site to "collect" all the work done during that course.

As in project 4 and 5, it combines html, css, javascript, and django to provide a seamless site. In this case images, css-files, and other necessary files are stored using django's whitenoise module. Mostly because users can not upload images themselves so every file can be considered essentially static in nature.

Also, this is the first project that uses Django5. Django5 is not really necessary for this project, but since it is available, why not?
