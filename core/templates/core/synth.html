#Synth.nu

 


This was the fifth and final project for the Code Institute course. It is very similar to the fourth project, except that it also adds credit card payments into the mix.

It combines html, css, javascript, django, and postgresSQL to provide an e-commerce site for vinyl-records and especially records belonging to the synth genre. Images, css-files, and other necessary files are stored at AWS.
