RetroLovers United

 


This was the fourth project for the Code Institute course, and the first one that can be considered full-stack.

It combines html, css, javascript, django, and postgresSQL to provide a community site for the subject at hand. Images, css-files, and other necessary files are stored at cloudinary.

Of course, if you are not interested in the subject (the venerable Amiga-line of computers) you will probably not find the content engaging.
