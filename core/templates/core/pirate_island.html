Pirate Island

 


This was the third project for the Code Institute course. For those that grew up with games from Infocom or Level9, it should be no surprise what this is.

For everyone else, it is a game that consists entirely of text and you control it by typing in commands such as "Go West", "Open Chest" and so on. The objective for such games is to get around all the obstacles in your way to get to the end scene.

It is called 'Interactive Fiction' for a reason.

The game is written entirely in python, with no external dependencies apart from python itself.

Note: some names and ideas were stolen from the Monkey Island series of games. They are not textbased, but otherwise the same ideas apply.
